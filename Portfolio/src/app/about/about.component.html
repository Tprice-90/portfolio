About Me

Terry Price Web Developer

I am a recent graduate of the Internet Applications and Web Development program at St Clair College. I began learning basic HTML, CSS and Javascript before taking the program but wanted to get a more formal education. Durring my studies was when I developed a true passion for web design. I learned the basics of both front-end and back-end development as well as how to work in different project management styles such as Agile, Waterfall and Hybrid. While I realized that I had a strong grasp of the front-end side of web development I decided to push myself to better understand the back-end by creating my own environment with LAMP and MEAN stacks hosted on a Virtual Machine with VirtualBox software. This helped me become more comfortable with dealing with database issues.
